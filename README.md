# Qiita Analysis

Qiitaチームで誰がどのくらい投稿したかを測定するツール

## Installation

```
go get github.com/kaneshin/qiita-analysis
```

_~/.giita_

```
{
  "team": "YOUR TEAM NAME",
  "token": "YOUR ACCESS TOKEN"
}
```

### Why `~/.giita`?

Compatible for [github.com/kaneshin/giita](https://github.com/kaneshin/giita).

## Usage

```
$ qiita-analysis
```
